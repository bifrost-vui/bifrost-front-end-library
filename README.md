## The Front-End library

We currently support 2 frameworks: **Angular and Twig**.
The Front-End library contains:
- **Components**
- **Organisms** (more complex components)
- **Templates** (to help build with components)
- **Pipes**
- **Styles** (e.g. Sass variables)
