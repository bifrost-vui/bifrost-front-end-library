# Icon Migration to Bifrost v1.3.0
Drupal only.

<br/><br/>

## New Icon Component API
**Twig Icon component** changes in Bifrost v1.3.0:
- The property **`type`** (`inertaface` or `illustration`) doesn't determine the size of the icon anymore.
- The property **`size`** is `small` by default when icon is interface type, and `medium` by default for type illustration.
- All `types` are available in all sizes and colors - make sure to follow [recommendations](https://www.figma.com/file/u25bSG8COM4azJ04CztTv8/DS---Medias?node-id=130%3A2093)
