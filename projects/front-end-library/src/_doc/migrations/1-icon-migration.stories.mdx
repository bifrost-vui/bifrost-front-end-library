### Other changes

All SVG files have been updated and now exist in large and small version ( naming convention : *filename-lg.svg* & *filename-sm.svg*)
Every Icon component now comes in 4 sizes : large, medium, small and xsmall.

For Drupal : there is no more Interface or Illustration directories.
- [see the Illustration icon list](#illustration-icon-list-json)
- [see the Interface icon list](#interface-icon-list-json)
