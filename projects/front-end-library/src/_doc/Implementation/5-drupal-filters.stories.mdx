# Twig filters list

<br/><br/>


## format_price

```jsx
Twig.extendFilter('format_price', function(price, options) {
    
    const language = (options && options[0]) || 'en';
    if (language === 'fr') {
        return price + '&nbsp;$';
    } else {
        return '$' + price;
    }
});
```

## json_parse

```jsx
Twig.extendFilter('json_parse', function(value) {
    try {
        return JSON.parse(value);
    } catch (e) {
        return value;
    }
});
```
