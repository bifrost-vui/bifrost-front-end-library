# Angular implementation
You have an **Angular installation** and you want to use Bifrost Design System.

<br/><br/>

- **Angular** version X or superior
- ...

## Architecture _(Migration)_
Structure your application to embrace Design System abstraction.
> Angular article: [Smart components vs Presentational components](https://blog.angular-university.io/angular-2-smart-components-vs-presentation-components-whats-the-difference-when-to-use-each-and-why/)

<br/>

## 1. Install the package
_TODO: Add Link to changelog version on Bitbucket._
```Shell
npm install bifrost-front-end-library --save
```


## 2. Import components
```js
import { ButtonModule } from 'bifrost-front-end-library';
```

## 3. Add a base styles
In `angular.json`:
```js
"styles": [
  "node_modules/bifrost-front-end-library/base.min.css",
  //...
],
```
Furthermore, you will be able to import variables in your sass files:
```css
@import 'node_modules/bifrost-front-end-library/source/globals/_variables.scss'

body {
    background-color: $primary;
}
```
