# Input-text Migration to Bifrost
Drupal only.

<br/><br/>

## New input-text Component API

### Naming chnages  only
Type | new name | old name | comment
----------------- | ----------------- | ----------------- | -----------------
string | inputId | id | -
string | inputValue | value | -
string | inputPlaceholder | placeholder | -
string | inputName | name | -
string | - | mtValue | depricated , add the bootstrap `mt-[1-6]` class to the `inputClass` to set the top margin on the input
string | errorMessage | invalidMsg | -

### naming and structural changes
Type | new name | old name | comment
----------------- | ----------------- | ----------------- | -----------------
object | label | label | the text can be passed as a string or an object  { text: string, class: string }
object | tooltip | labelTooltipTxt | the text can be passed as a string or an object { text: string, position: top,bottom,left or right }
object | info| subInfos | the text can be passed as a string or an object { text: string, url: string }
