# Colors

Bifr√∂st defines 5 groups of colors: **Content, Background, Background Action, Stroke** and **Feedback**.

These colors has a default color and some of them has a `reversed` color.

The `reversed` color will be automatically used if the component or its parent has the `.reversed` class.
We can call that the _**reversed mode**_.

If there is no `reversed` color, the default color will be used instead in the `reversed` mode.

## CSS Variables

CSS variables are available on the `:root` of the CSS file. They can be used in any CSS property
that contains a color value.

An example would be: `color: var(--bf-color-highlight);`.

If a color has a `reversed` value, a CSS variable with the suffix `-reversed` will be available.
It can be useful to have access to the `reversed` value of a color when in _**default mode**_
and the same thing in _**reversed mode**_.

### Values of variables in _Default Mode_ and in _Reversed Mode_

Say that we take the variables `--bf-color-highlight` and `--bf-color-highlight-reversed`.

In the **default mode**:
- the value of `--bf-color-highlight` is **`#050504`**;
- the value of `--bf-color-highlight-reversed` is **`#ffd200`**.

In the **reversed mode**:
- the value of `--bf-color-highlight` is **`#ffd200`**;
- the value of `--bf-color-highlight-reversed` is **`#050504`**.
