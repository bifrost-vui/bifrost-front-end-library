# Headings

**Styles:** The style applies with only a heading tag, e.g. : `<h1>My title</h1>`.
However, you would apply a heading style to a different semantic tag, e.g: `<h1 class='h2'>My title</h1>`.

**Theme:** If you are not using the Videotron default theme, make sure you are wrapping your code around a parent with parameter `data-theme='<your-theme-name>'`. See [Theme documentation](?path=/docs/implementation-theme--page).
