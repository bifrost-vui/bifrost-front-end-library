# Showcase Drupal

## Install Showcase Drupal

<br/>

### 1. Install Drupal
```
cd projects/showcase-drupal
curl -sSL https://www.drupal.org/download-latest/tar.gz | tar -xz --strip-components=1
```

### 2. Launch Drupal
```
php ./core/scripts/drupal quick-start bifrost
```
The front part of the website will not work, because you still have to install Module Component (next step).
However, you should be able to access Drupal Administration from [http://127.0.0.1:8888/admin/](http://127.0.0.1:8888/admin/).

### 3. Install Module Components
See [Module documentation](https://www.drupal.org/project/components).
```
composer require 'drupal/components:^2.4'
```

Go to [http://127.0.0.1:8888/admin/modules](http://127.0.0.1:8888/admin/modules) and enable Component Module in Drupal Administration.
You should see the confirmation message: **Module Components has been enabled.**

### 4. Install Bifrost Package
```
cd themes/bifrost && npm install
```

### 5. Relaunch Drupal
```
php ./core/scripts/drupal quick-start bifrost
```
There you go.
