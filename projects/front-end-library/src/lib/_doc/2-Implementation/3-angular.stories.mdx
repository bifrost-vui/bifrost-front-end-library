# Angular implementation
You have an **Angular installation**, and you want to use Bifröst Design System.

<br/><br/>


## 1. Install the Bifröst package
[Package NPM Changelog](https://bifrost-front-end-library-vui.vercel.app/?path=/docs/changelog--page)
```bash
npm install @videotron-ui/bifrost-front-end-library --save
```


## 2. Add styles
Add **bifrost.css** in your **angular.json** application:
```ts
"styles": [
  "node_modules/@videotron-ui/bifrost-front-end-library/assets/css/bifrost.css",
  //...
],
```


## 3. Import components
**Note:** Currently, only the component **Badge** is exported:
[Badge API](https://bifrost-front-end-library-vui.vercel.app/?path=/story/components-badge--angular)
```ts
import { NgModule }             from '@angular/core';
import { BadgeModule }          from '@videotron-ui/bifrost-front-end-library';

@NgModule({
    declarations: [],
    imports: [
        BadgeModule,
    ],
    providers: [],
    bootstrap: []
})
export class YourModule { }
```

Use it in your template (note that all components tags are prefix with **bf-**).
```html
<bf-badge label='My Badge' hierarchy='primary' className='mt-3'></bf-badge>
```



## 4. Learn about the foundation
For example, you could start with the [System of Colors](https://bifrost-front-end-library-vui.vercel.app/?path=/docs/styles-colors--page). Then explore the [Theme](https://bifrost-front-end-library-vui.vercel.app/?path=/docs/implementation-theme--page) and [Reversed](https://bifrost-front-end-library-vui.vercel.app/?path=/docs/implementation-reversed-highlight--page) documentation.
Feel free to explore all pages in **IMPLEMENTATION** and **STYLES** sections.



## 5. Learn about Components API and documentation
Select any component on the sidebar, you will see tabs at the very top:
- **Docs tab** displays complete documentation with all component variations in a single page.
- **Canvas tab**: Please, select **Controls panel** and explore the (interactive) API documentation.
