# Input Slider

## Component API

**âœ… Inputs are equivalent for Angular and Drupal.**

<Props of={InputSliderComponent} />

### Notes
- if `value` is empty or undefined : creates a 2 handles slider positioned at each end of the input track
- if `value` contains 2 values : creates a 2 handles slider positioned at the corresponding coordonates
- if `value` contains 1 value : creates a 1 handle slider positioned  at the corresponding coordonate
- if `isInverted` is set true and `value` contains 1 value : reverts the direction of the 1 handle slider but has no effect if `value` contains 2 values
