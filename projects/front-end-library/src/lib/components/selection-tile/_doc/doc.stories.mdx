## More Info


- The *Selection Tile* comes in **horizontal** and **vertical** orientation, each having a **small** and a **large** version.
  - size(default -> small) 
  - orientation(default -> horizontal)
  - **horizontal** version won't display images
- The Tile mimics the behavior of the input type :
  - mutual exclusion if it's a **radio button**
  - multi select in case of a **checkbox**
- The code within the tile_content editable block manages most common tile variants. Although it's not mandatory, it's recommended to use as is.
- Upon redefinition within an embed Twig block, be aware that items **won't be clickable**.
- If you need the content to be clickable, then use the **Tile component** instead.
