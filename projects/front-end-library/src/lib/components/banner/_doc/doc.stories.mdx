## Drupal implementation example

#### With icon
```jsx
{% include '@bf-components/banner/twig/index.twig' with {
    href        : 'https://videotron.com',

    paragraph   : 'Free with any purchase of this phone, valuted at $239.38.',

} only %}
```

#### With image
```jsx
{% include '@bf-components/banner/twig/index.twig' with {

        alt: 'Alt description'
    },
} only %}
```
