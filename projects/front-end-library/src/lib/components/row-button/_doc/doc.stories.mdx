## Drupal implementation example

#### List of Row Buttons

Wrap your <b>Row Buttons</b> with CSS classes `.bf-list .bf-list--divider` in order to display dividers between them.

```jsx
<ul class='bf-list bf-list--divider'>
    <li class='w-100'>
        {% include '@bf-components/row-button/twig/index.twig' with {
            href        : 'https://example.com',
            title       : 'Private viewing',
            paragraph   : 'Contribute to the ambiance of your spaces.',
            class       : 'mb-4',
        } only %}
    </li>
</ul>
```

#### Row Button only
```jsx
{% include '@bf-components/row-button/twig/index.twig' with {
    href        : 'https://example.com',
    title       : 'Private viewing',
    paragraph   : 'Contribute to the ambiance of your spaces.',
    class       : 'mb-4',
} only %}
```

#### Row Button with icon
```jsx
{% include '@bf-components/row-button/twig/index.twig' with {
    // ...
    iconName    : 'mobile-multiline',
} only %}
```

#### Row Button with image
