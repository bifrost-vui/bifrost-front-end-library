# Header

> This component is the same as the old **Navigation Header**.
It is renamed as **Navigation Header Business** because we still need
to support the [Business](https://videotron.com/en/business) section
of the website. The new [Navigation Header](/story/components-navigation-header--drupal)
component will only support the [Residential](https://videotron.com/en) section at first.
Eventually, in a second phase, the new component will integrate
the functionalities of the **Business** one.

<br />
