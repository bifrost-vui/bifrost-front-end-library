## Drupal implementation example

> Note: You don't have to specify if the icon is **interface** or **illustration** [more details](?path=/docs/styles-icons--page).

### Interface Icons

> NOTE : the icon size should be `small` or `xsmall`

```jsx
{% include '@bf-components/icon/twig/index.twig' with {
    name: 'edit',
    size: 'small'
    color: 'negative',
    class: 'mr-2'
} only %}

```

### Illsutration Icons

> NOTE : the icon size should be `small`, `medium` or `large`

```jsx
{% include '@bf-components/icon/twig/index.twig' with {
    name: 'my-account',
    size: 'large',
} only %}

```

### Illsutration Huge Icons

> NOTE : the icon size `huge` should only be used for hero components

```jsx
{% include '@bf-components/icon/twig/index.twig' with {
    name: 'my-account',
    size: 'huge',
} only %}

```
