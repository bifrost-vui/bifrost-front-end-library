# Progress

This component has been migrated from the UI-Kit project without any optimisation

**Progress bar Colors**

- when `progressPercentValue` > 90 : color = danger (red)
- when `progressPercentValue` > 50 : color = warning (orange)
- when `progressPercentValue` =< 50 : color = success (green)
- when `isMonochromeColor` is true, the progression is always yellow, no regards to the progression percentage
