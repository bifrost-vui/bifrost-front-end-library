# Stepper

 This component enables the end user to visualise his progression when he's going trough a multi-step process.
 - It's generaly used in combinaition with a `Funnel` type navigation header.
 - It's navigation capacities are limited to the previous step and next step within the current process
 - If `previousHasLabel` and/or `nextHasLabel` are set to false, a tooltip wil display the step label bellow the arrow icon link
