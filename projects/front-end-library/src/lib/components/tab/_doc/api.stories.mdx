<h3 id="inavitems-format">Expected format</h3>

#### Tab-like Nav

```jsx
{
    "class": "active",
    "extraAttribute": {
        "data-example-1": "exampleValue-1",
        "data-example-2": "exampleValue-2"
    },
    "href": "/channelSelection",
    "label": "Channel selection"
}
```

#### Real tab functionality

```jsx
{
    "ariaControls": "tabContentChannel",
    "ariaSelected": "true",
    "class": "active",
    "dataTarget": "tabContentChannel",
    "extraAttribute": {
        "data-example-1": "exampleValue-1",
        "data-example-2": "exampleValue-2"
    },
    "id": "tabChannel",
    "label": "Channel selection"
}
```
