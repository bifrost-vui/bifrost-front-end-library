# Input Search Business

> This component is the same as the old **Input Search**.
It is renamed as **Input Search Business** because we still need
to support the [Business](https://videotron.com/en/business) section
of the website. The new [Input Search](/story/components-input-search--drupal)
component will only support the [Residential](https://videotron.com/en) section at first.
Eventually, in a second phase, when the [Navigation Header Business](/story/components-navigation-header-business--drupal)
component functionalities will be integrated in the new [Navigation Header](/story/components-navigation-header--drupal),
this **Input Search Business** component will be deprecated and unused.
