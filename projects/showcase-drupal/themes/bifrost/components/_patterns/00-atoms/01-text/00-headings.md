### HTML tags _-VS-_ CSS class

Bootstrap propose two methods for styling heading text as headers. They both look the same _BUT!_

1. The **real HTML** tags in the DOM, from <code>&lt;h1&gt;</code> to <code>&lt;h6&gt;</code> are for SEO (_Search Engine Optimisation_). There for these tags are to be used based on the **semantic** of the HTML document.
2. The **CSS** class from <code>&lt;* class="h1"&gt;</code> to <code>&lt;* class="h6"&gt;</code> that can be applied on any HTML tag to match **design** requirements.
